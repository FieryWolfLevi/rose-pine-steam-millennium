## Usage

1. Install the [Millennium Patcher](https://steambrew.app/) for Steam.
2. Place this theme folder inside your Millennium skins directory:
   `P:\Program Files (x86)\Steam\millennium\themes\rose-pine`
3. Launch Steam, click on the **Steam** menu in the top-left, and select **Millennium**.
4. In the settings panel, go to **Themes**, select **Rosé Pine**, and choose your preferred variant:
   - **Rosé Pine** (Main dark variant)
   - **Rosé Pine Moon** (Alternative dark variant)
   - **Rosé Pine Dawn** (Light variant)
5. Select the light/dark option under **Appearance** to toggle between dark modes and light modes.
6. Click apply and restart Steam.
